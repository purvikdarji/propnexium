# PropNexium Views

JSP views directory. Place all .jsp files here.

Structure:
- home.jsp         — Landing page
- login.jsp        — Login form
- register.jsp     — Registration form
- dashboard.jsp    — User dashboard
- properties/
  - list.jsp       — Property listings grid
  - detail.jsp     — Property detail page
  - search.jsp     — Advanced search page
  - new.jsp        — Create property form
  - edit.jsp       — Edit property form
- admin/
  - index.jsp      — Admin dashboard
- agent/
  - index.jsp      — Agent dashboard
